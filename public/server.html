O que esta página responde

Mostra se o proxy está recebendo chamadas, transformando payloads e entregando respostas úteis para os apps/usuários do ecossistema VALORAE.

Como interpretar

Priorize respostas entregues, taxa render/cache safe, latência e payloads. Se tudo estiver parcial, consulte Qualidade dos dados antes de culpar o app.

Cada item representa uma resposta capturada pelo monitor de saída. Chamadas internas de polling são isoladas para não poluir as métricas de usuários.

Uso recomendado

Qualidade dos dados

Une consistência de campos, orçamento de payload, maturidade do Engine, cobertura por Ação/FII, cache e plano de ação em uma visão única.

Por que existe

Dados financeiros podem vir parciais, divergentes ou em formatos diferentes. Esta página separa problema de fonte, peso de payload e segurança de renderização.

Como usar

Leia primeiro render/cache safe. Depois confira consistência, orçamento do payload, cobertura por classe de ativo e plano de ação recomendado.
