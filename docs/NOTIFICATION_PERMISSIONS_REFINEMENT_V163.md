# Proxy v163 — Radar de notícias para notificações

- Amplia `symbols` aceitos pelo endpoint de notícias para até 96 tickers.
- Reforça a consulta de carteira com termos de evento: fato relevante, comunicado ao mercado, CVM, proventos, resultado, rating, recompra e subscrição.
- Adiciona `notificationCandidate`, `relevanceScore` e `relevanceSignals` aos itens normalizados para apoiar o APK em alertas mais acionáveis.
- Mantém fallback de busca do Google Notícias quando o RSS público não entrega itens recentes.
